### 使用：
注意，vscode的终端有问题，使用Docker总是permission deny，还是得用自己电脑的终端。

```
docker build     --build-arg CUDA_VERSION=11.8.0     --build-arg CUDA_ARCHITECTURES=70     --build-arg OS_VERSION=22.04     --tag nerfstudio-86     --file Dockerfile .
```
